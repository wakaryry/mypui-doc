# mypUI 中文文档

文档地址：[mypui](https://mypui.redou.vip/)

该站点基于 [Hexo](https://hexo.io/) 和 [Vue中文文档](https://cn.vuejs.org/) 构建而成。网站内容在 `src` 文件夹内，格式为 Markdown。欢迎 issue 或 pull request。

## 开发

``` bash
$ npm install
$ npm start # 开发服务器地址为 http://localhost:4000
```
